# ステップ3: 商品登録

nancoで在庫管理を始めるには、まず商品を登録する必要があります。

## 商品登録の基本手順

### 1. 商品管理画面にアクセス

1. 左メニューから「商品管理」を選択
2. 「新規商品登録」ボタンをクリック

### 2. 基本情報の入力

以下の情報を入力してください：

#### 必須項目
- **商品名**: 商品の名前（例：りんご、トイレットペーパー）
- **商品コード（SKU）**: 商品を識別するためのコード（例：APPLE001、TP001）
- **カテゴリ**: 事前に設定したカテゴリから選択
- **仕入れ価格**: 商品の仕入れ価格
- **販売価格**: 商品の販売価格
- **初期在庫数**: 現在の在庫数

#### オプション項目
- **説明**: 商品の詳細説明
- **仕入れ先**: 商品の仕入れ先
- **最小在庫数**: 在庫不足アラートの閾値
- **単位**: 個、箱、kg、Lなど

### 3. 商品画像のアップロード

商品画像をアップロードすることで、視覚的に商品を管理できます：

- 推奨サイズ: 800x600px以上
- 対応形式: JPG, PNG, WebP
- 最大ファイルサイズ: 5MB

### 4. 商品の保存

すべての情報を入力後、「保存」ボタンをクリックして商品を登録します。

## 一括登録（CSVインポート）

多数の商品を一度に登録する場合は、CSVファイルを使用できます：

### 1. CSVファイルの準備

以下の形式でCSVファイルを作成：

```csv
商品名,商品コード,カテゴリ,仕入れ価格,販売価格,初期在庫数
りんご,APPLE001,食品,100,150,50
トイレットペーパー,TP001,日用品,200,300,30
```

### 2. CSVファイルのアップロード

1. 「商品管理」→「CSVインポート」を選択
2. CSVファイルをアップロード
3. データの確認とマッピング
4. 「インポート実行」をクリック

## 商品の編集・削除

### 商品情報の編集

1. 商品一覧から編集したい商品を選択
2. 「編集」ボタンをクリック
3. 必要に応じて情報を変更
4. 「保存」をクリック

### 商品の削除

1. 商品一覧から削除したい商品を選択
2. 「削除」ボタンをクリック
3. 確認ダイアログで「削除」をクリック

**注意**: 在庫がある商品は削除できません。先に在庫を0にしてから削除してください。

## 商品検索・フィルター

### 検索機能

- 商品名での検索
- 商品コードでの検索
- カテゴリでの絞り込み
- 在庫数での絞り込み

### ソート機能

- 商品名順
- 在庫数順
- 登録日順
- 価格順
