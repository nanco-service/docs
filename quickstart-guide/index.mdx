# 





### 🚀 Aコース：お手軽コース
**「とにかく、一番手軽に始めたい」**

このスタートガイドは、こんな方に向いています：
- とにかく少ない手間で、直ぐに使い始めたい
- 扱うアイテムは20～50品程
- 管理するのは基本的に１人（後で追加できればOK）

「いちばん基本的な操作が出来ればとりあえずOK、細かいことは後からゆっくり覚えます」そんなタイプの人はこの方法でスタートしてみましょう。


**このコースの内容：**
- 商品（アイテム）の個別追加
- 属性項目の追加（価格、メーカーなど）
- 一括でのアイテム追加（CSV/Excel）
- 基本的な在庫管理操作

### ⚡ Bコース：バーコード棚卸コース
**「棚卸をクイックに頻繁に実施したい。できるだけ在庫数のズレを無くしたい。」**

このスタートガイドは、こんな方に向いています：
- 棚卸を頻繁に行う
- SKUは150種類～500種類ほど
- 数名で管理

**準備：**
- エクセルやCSVなど商品のデータ
- 商品のバーコードの情報（あるとベター）

管理する商品の種類が数百くらいあって、エクセルで管理するときも探すのが大変、という方、棚卸を頻繁にやる方はこの方法で進めていきましょう。

**このコースの内容：**
- 商品（アイテム）の個別追加
- 属性項目の追加（価格、メーカーなど）
- 一括でのアイテム追加（CSV/Excel）
- メンバー追加とチーム管理
- スマートフォンアプリの活用
- バーコード機能の活用

### 🎯 Cコース：写真重視コース
**「一品モノも多いので写真が大事。管理数もとても多い…！！」**

このスタートガイドは、こんな方に向いています：
- 商品の識別を外観に頼ることが多い
- 1000種類以上
- 多数のスタッフで管理（10名）
- LINE等グループツールで報告用に商品画像をシェアしたい

条件：
- 商品のエクセルデータがある

管理する商品が1000点を越えていて、なおかつ、商品画像の登録が必須、という場合はこの方法でスタートしましょう。

**このコースの内容：**
- 商品（アイテム）の個別追加
- 一括でのアイテム追加（CSV/Excel）
- メンバー追加とチーム管理
- スマートフォンアプリの活用
- 商品写真の登録（PC・スマホ）
- グリッド表示での商品管理

**所要時間：** 約1-1.5時間
