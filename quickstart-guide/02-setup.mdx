# ステップ2: 初期設定

アカウント作成後、nancoを効率的に使用するための初期設定を行います。

## ダッシュボードの確認

nancoにログインすると、まずダッシュボードが表示されます。ダッシュボードでは以下の情報を確認できます：

- **在庫サマリー**: 現在の在庫状況の概要
- **最近の活動**: 最近の入出庫履歴
- **アラート**: 在庫不足や期限切れの警告
- **クイックアクション**: よく使う機能へのショートカット

## 基本設定

### 1. 会社情報の設定

1. 左メニューから「設定」→「会社情報」を選択
2. 以下の情報を入力：
   - 会社名（店舗名）
   - 住所
   - 電話番号
   - 業種
3. 「保存」をクリック

### 2. 通貨設定

1. 「設定」→「通貨設定」を選択
2. 使用する通貨を選択（JPY、USD、EURなど）
3. 小数点以下の桁数を設定
4. 「保存」をクリック

### 3. 在庫単位の設定

1. 「設定」→「在庫単位」を選択
2. 使用する単位を追加：
   - 個、箱、kg、Lなど
3. 「保存」をクリック

## カテゴリの設定

### 基本カテゴリの作成

1. 「設定」→「カテゴリ管理」を選択
2. 「新規カテゴリ作成」をクリック
3. 以下のカテゴリを作成：
   - 食品
   - 日用品
   - 衣類
   - その他

### 階層構造の作成（オプション）

必要に応じて、より詳細なカテゴリ構造を作成できます：

```
食品
├── 生鮮食品
│   ├── 野菜
│   └── 果物
└── 加工食品
    ├── 調味料
    └── スナック
```

## ユーザーの招待

### チームメンバーの追加

1. 「設定」→「ユーザー管理」を選択
2. 「ユーザーを招待」をクリック
3. 以下の情報を入力：
   - メールアドレス
   - 氏名
   - 権限レベル（オペレーター推奨）
4. 「招待メールを送信」をクリック
