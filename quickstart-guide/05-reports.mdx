# ステップ5: レポート活用

nancoのレポート機能を使用して、在庫状況や売上データを分析し、効率的な在庫管理を実現します。

## 基本的なレポート

### 在庫レポート

現在の在庫状況を把握するためのレポートです：

1. 「レポート」→「在庫レポート」を選択
2. 以下の情報を確認：
   - 全商品の現在在庫数
   - 在庫価値の合計
   - 在庫不足商品の一覧
   - 在庫過多商品の一覧

### 入出庫レポート

入出庫の履歴を分析するレポートです：

1. 「レポート」→「入出庫レポート」を選択
2. 期間を指定してレポートを生成
3. 以下の情報を確認：
   - 期間別の入庫実績
   - 期間別の出庫実績
   - 商品別の入出庫傾向

### 売上レポート

売上データを分析するレポートです：

1. 「レポート」→「売上レポート」を選択
2. 期間と商品を指定
3. 以下の情報を確認：
   - 期間別の売上実績
   - 商品別の売上分析
   - カテゴリ別の売上分析

## レポートの活用例

### 在庫不足の特定

1. 「在庫レポート」を開く
2. 「在庫不足」タブを選択
3. 最小在庫数を下回っている商品を確認
4. 発注が必要な商品を特定

### 売れ筋商品の分析

1. 「売上レポート」を開く
2. 期間を指定（例：過去1ヶ月）
3. 売上順でソート
4. 売れ筋商品を特定

### 在庫回転率の確認

1. 「在庫回転率レポート」を開く
2. 商品別の回転率を確認
3. 回転率の低い商品を特定
4. 在庫削減の検討

## レポートのエクスポート

### PDFレポート

印刷や共有に適したPDF形式でエクスポート：

1. レポート画面で「エクスポート」をクリック
2. 「PDF」を選択
3. ファイル名を入力
4. 「ダウンロード」をクリック

### Excelレポート

データの詳細分析に適したExcel形式でエクスポート：

1. レポート画面で「エクスポート」をクリック
2. 「Excel」を選択
3. ファイル名を入力
4. 「ダウンロード」をクリック

## 定期レポートの設定

### 自動レポートの設定

定期的にレポートを自動生成できます：

1. 「設定」→「定期レポート」を選択
2. 「新規定期レポート作成」をクリック
3. 以下の項目を設定：
   - レポート名
   - レポートの種類
   - 実行頻度（日次、週次、月次）
   - 実行時刻
   - 送信先（メールアドレス）
4. 「保存」をクリック

### 定期レポートの管理

- 定期レポート一覧の表示
- 定期レポートの編集・削除
- 実行履歴の確認
- 一時停止・再開

## ダッシュボードの活用

### カスタムダッシュボード

重要な指標を一画面に表示：

1. 「ダッシュボード」→「カスタム」を選択
2. 表示したいレポートを選択
3. レイアウトを調整
4. 「保存」をクリック

### リアルタイム更新

ダッシュボードはリアルタイムで更新されます：

- 在庫数の変更
- 新規入出庫の反映
- アラートの表示
