# nancoの基本操作

nancoは、シンプルで使いやすい在庫管理クラウドサービスです。このガイドでは、nancoの基本的な操作について説明します。

## ダッシュボードの概要

nancoにログインすると、まずダッシュボードが表示されます。ダッシュボードでは以下の情報を確認できます：

- **在庫サマリー**: 現在の在庫状況の概要
- **最近の活動**: 最近の入出庫履歴
- **アラート**: 在庫不足や期限切れの警告
- **クイックアクション**: よく使う機能へのショートカット

## 商品の登録

### 1. 商品情報の入力

1. 左メニューから「商品管理」を選択
2. 「新規商品登録」ボタンをクリック
3. 以下の情報を入力：
   - 商品名
   - 商品コード（SKU）
   - カテゴリ
   - 仕入れ価格
   - 販売価格
   - 初期在庫数

### 2. 商品画像のアップロード

商品画像をアップロードすることで、視覚的に商品を管理できます：

- 推奨サイズ: 800x600px以上
- 対応形式: JPG, PNG, WebP
- 最大ファイルサイズ: 5MB

## 在庫の管理

### 入庫処理

1. 「入庫管理」メニューを選択
2. 「新規入庫」をクリック
3. 商品を選択し、入庫数を入力
4. 入庫日と備考を記入
5. 「入庫確定」をクリック

### 出庫処理

1. 「出庫管理」メニューを選択
2. 「新規出庫」をクリック
3. 商品を選択し、出庫数を入力
4. 出庫理由を選択（販売、廃棄、移動など）
5. 「出庫確定」をクリック

## 在庫の確認

### 在庫一覧の表示

- 「在庫管理」メニューから在庫一覧を確認
- フィルター機能で商品名、カテゴリ、在庫数で絞り込み可能
- ソート機能で在庫数、商品名、更新日で並び替え可能

### 在庫アラートの設定

在庫が一定数以下になった時にアラートを受け取ることができます：

1. 商品詳細画面で「アラート設定」をクリック
2. 最小在庫数を設定
3. 通知方法を選択（メール、プッシュ通知）

## レポートの活用

nancoでは様々なレポートを生成できます：

- **在庫レポート**: 現在の在庫状況
- **入出庫レポート**: 期間別の入出庫履歴
- **売上レポート**: 商品別の売上分析
- **在庫回転率レポート**: 商品の回転率分析
