# ユーザー権限の管理

nancoでは、複数のユーザーが同じアカウントで在庫管理を行うことができます。このガイドでは、ユーザーの追加と権限の管理について説明します。

## ユーザー権限の種類

### 1. 管理者（Admin）

すべての機能にアクセスできる最高権限です：

- ユーザーの追加・削除・編集
- システム設定の変更
- すべてのデータの閲覧・編集
- レポートの生成・エクスポート
- アカウント設定の変更

### 2. マネージャー（Manager）

日常的な業務管理に必要な権限です：

- 商品の追加・編集・削除
- 入出庫処理
- 在庫アラートの設定
- レポートの閲覧・生成
- カテゴリの管理

### 3. オペレーター（Operator）

基本的な業務操作に必要な権限です：

- 商品の閲覧
- 入出庫処理
- 在庫の確認
- 基本的なレポートの閲覧

### 4. 閲覧者（Viewer）

データの閲覧のみ可能な権限です：

- 商品の閲覧
- 在庫の確認
- 基本的なレポートの閲覧
- データの編集は不可

## ユーザーの追加

### 新規ユーザーの招待

1. 「設定」→「ユーザー管理」を選択
2. 「ユーザーを招待」ボタンをクリック
3. 以下の情報を入力：
   - メールアドレス
   - 氏名
   - 権限レベル
   - 部署・役職（任意）
4. 「招待メールを送信」をクリック

### 招待メールの処理

招待されたユーザーは以下の手順でアカウントを作成します：

1. 招待メールのリンクをクリック
2. パスワードを設定
3. プロフィール情報を入力
4. アカウントの有効化

## ユーザー権限の変更

### 権限レベルの変更

1. 「設定」→「ユーザー管理」を選択
2. 対象ユーザーの「編集」ボタンをクリック
3. 権限レベルを変更
4. 「保存」をクリック

### 権限の詳細設定

各権限レベルで以下の詳細設定が可能です：

- **商品管理**: 商品の追加・編集・削除
- **入出庫管理**: 入出庫処理の実行
- **在庫管理**: 在庫の確認・調整
- **レポート**: レポートの生成・エクスポート
- **設定**: システム設定の変更
- **ユーザー管理**: ユーザーの追加・編集・削除

## ユーザーの管理

### ユーザー一覧の表示

1. 「設定」→「ユーザー管理」を選択
2. ユーザー一覧で以下の情報を確認：
   - 氏名
   - メールアドレス
   - 権限レベル
   - 最終ログイン日時
   - アカウント状態

### ユーザーの検索・フィルター

- 名前での検索
- 権限レベルでの絞り込み
- 部署での絞り込み
- アカウント状態での絞り込み

### ユーザーの無効化

1. ユーザー一覧から対象ユーザーを選択
2. 「無効化」ボタンをクリック
3. 確認ダイアログで「無効化」をクリック

**注意**: 無効化されたユーザーはログインできなくなりますが、データは保持されます。

## セキュリティ設定

### パスワードポリシー

nancoでは以下のパスワードポリシーを適用しています：

- 最小8文字以上
- 大文字・小文字・数字・記号を含む
- 90日ごとのパスワード変更
- 過去5回のパスワードとの重複禁止

### 二段階認証（2FA）

セキュリティを向上させるために二段階認証を設定できます：

1. 「設定」→「セキュリティ」を選択
2. 「二段階認証を有効にする」をクリック
3. 認証アプリでQRコードをスキャン
4. 認証コードを入力して設定完了

### セッション管理

- 同時ログイン数の制限
- セッションタイムアウトの設定
- ログイン履歴の確認
- 異常なログインの検知

## アクティビティログ

### ログの確認

ユーザーの操作履歴を確認できます：

1. 「設定」→「アクティビティログ」を選択
2. 以下の情報を確認：
   - 操作日時
   - ユーザー名
   - 操作内容
   - IPアドレス
   - ブラウザ情報

### ログのフィルター

- 期間での絞り込み
- ユーザーでの絞り込み
- 操作内容での絞り込み
- 重要度での絞り込み

## ベストプラクティス

### 権限の最小化

- 必要最小限の権限を付与
- 定期的な権限の見直し
- 不要な権限の削除

### ユーザー管理

- 退職者のアカウントを即座に無効化
- 定期的なユーザー一覧の確認
- 権限変更の記録保持

### セキュリティ

- 強力なパスワードの使用
- 二段階認証の有効化
- 定期的なセキュリティ監査

## トラブルシューティング

### よくある問題

**Q: ユーザーがログインできない**
A: アカウントが無効化されていないか、パスワードが正しいか確認してください。

**Q: 権限が変更されない**
A: 管理者権限が必要です。現在の権限レベルを確認してください。

**Q: 招待メールが届かない**
A: スパムフォルダを確認し、メールアドレスが正しいか確認してください。
