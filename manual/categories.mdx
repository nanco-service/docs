# 商品カテゴリの管理

商品カテゴリを適切に設定することで、商品の整理と検索が効率的になります。このガイドでは、nancoでのカテゴリ管理について説明します。

## カテゴリの作成

### 基本カテゴリの設定

1. 左メニューから「設定」→「カテゴリ管理」を選択
2. 「新規カテゴリ作成」ボタンをクリック
3. 以下の情報を入力：
   - カテゴリ名
   - カテゴリコード（任意）
   - 説明
   - 親カテゴリ（階層構造を作る場合）

### 階層構造の活用

nancoでは最大3階層までのカテゴリ構造を作成できます：

```
例：
├── 食品
│   ├── 生鮮食品
│   │   ├── 野菜
│   │   └── 果物
│   └── 加工食品
│       ├── 調味料
│       └── スナック
└── 日用品
    ├── 洗剤
    └── 紙製品
```

## カテゴリの編集

### カテゴリ情報の変更

1. カテゴリ一覧から編集したいカテゴリを選択
2. 「編集」ボタンをクリック
3. 必要に応じて情報を変更
4. 「保存」をクリック

### カテゴリの削除

**注意**: 商品が登録されているカテゴリは削除できません

1. カテゴリ一覧から削除したいカテゴリを選択
2. 「削除」ボタンをクリック
3. 確認ダイアログで「削除」をクリック

## 商品へのカテゴリ割り当て

### 新規商品登録時

商品登録時にカテゴリを選択できます：

1. 商品登録画面で「カテゴリ」フィールドをクリック
2. ドロップダウンから適切なカテゴリを選択
3. 階層構造の場合は、親カテゴリ→子カテゴリの順で選択

### 既存商品のカテゴリ変更

1. 商品一覧から対象商品を選択
2. 「編集」ボタンをクリック
3. カテゴリを変更
4. 「保存」をクリック

## カテゴリ別レポート

### 在庫レポート

カテゴリ別の在庫状況を確認できます：

1. 「レポート」→「在庫レポート」を選択
2. フィルターでカテゴリを指定
3. レポートを生成

### 売上レポート

カテゴリ別の売上分析も可能です：

1. 「レポート」→「売上レポート」を選択
2. 期間とカテゴリを指定
3. 売上データを分析

## カテゴリのベストプラクティス

### 命名規則

- 分かりやすい名前を使用
- 一貫性のある命名規則を採用
- 略語は避ける

### 階層構造の設計

- 深すぎる階層は避ける（最大3階層）
- 関連する商品をグループ化
- 将来的な拡張を考慮

### 定期的な見直し

- 月1回程度カテゴリ構造を見直す
- 使用されていないカテゴリを整理
- 新しい商品に対応したカテゴリを追加

## 一括操作

### カテゴリの一括変更

複数の商品のカテゴリを一度に変更できます：

1. 商品一覧で対象商品を複数選択
2. 「一括操作」→「カテゴリ変更」を選択
3. 新しいカテゴリを指定
4. 「適用」をクリック

### カテゴリのインポート/エクスポート

CSVファイルを使用してカテゴリ情報を一括で管理できます：

1. 「設定」→「カテゴリ管理」→「インポート/エクスポート」
2. CSVファイルをアップロードまたはダウンロード
3. データを編集して再アップロード
