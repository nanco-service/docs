# 在庫アラートの設定

nancoの在庫アラート機能を使用することで、在庫が不足する前に事前に通知を受け取ることができます。このガイドでは、アラートの設定方法について説明します。

## アラートの種類

nancoでは以下の種類のアラートを設定できます：

### 1. 在庫不足アラート
- 在庫数が設定した閾値を下回った時に通知
- 商品別に個別設定可能

### 2. 在庫過多アラート
- 在庫数が設定した上限を超えた時に通知
- 過剰在庫の防止に役立つ

### 3. 期限切れアラート
- 商品の賞味期限や消費期限が近づいた時に通知
- 食品や医薬品の管理に有効

### 4. 入庫予定アラート
- 入庫予定日の前に通知
- 入庫の準備を忘れないように

## アラートの設定方法

### 商品別アラート設定

1. 商品一覧から対象商品を選択
2. 「アラート設定」タブをクリック
3. 以下の項目を設定：
   - **最小在庫数**: 在庫不足アラートの閾値
   - **最大在庫数**: 在庫過多アラートの閾値
   - **期限切れ通知日数**: 期限切れの何日前に通知するか
   - **通知方法**: メール、プッシュ通知、SMS

### 一括アラート設定

複数の商品に同じアラート設定を適用できます：

1. 商品一覧で対象商品を複数選択
2. 「一括操作」→「アラート設定」を選択
3. 共通のアラート条件を設定
4. 「適用」をクリック

## 通知方法の設定

### メール通知

1. 「設定」→「通知設定」を選択
2. 「メール通知」タブをクリック
3. 通知を受け取るメールアドレスを登録
4. 通知頻度を設定（即座、日次、週次）

### プッシュ通知

1. 「設定」→「通知設定」を選択
2. 「プッシュ通知」タブをクリック
3. ブラウザの通知許可を有効化
4. 通知音の設定

### SMS通知

1. 「設定」→「通知設定」を選択
2. 「SMS通知」タブをクリック
3. 携帯電話番号を登録
4. 通知の有効/無効を設定

## アラート履歴の確認

### アラート一覧の表示

1. 「アラート」メニューを選択
2. アラート一覧で過去の通知を確認
3. フィルター機能で期間や種類で絞り込み

### アラートの詳細確認

1. アラート一覧から詳細を確認したい項目を選択
2. アラートの詳細情報を表示
3. 対応状況を更新

## アラートのベストプラクティス

### 適切な閾値の設定

- **最小在庫数**: 通常の販売量を考慮して設定
- **最大在庫数**: 保管スペースとコストを考慮
- **期限切れ通知**: 商品の特性に応じて設定

### 通知頻度の調整

- 重要な商品: 即座通知
- 一般的な商品: 日次通知
- 季節商品: 週次通知

### 定期的な見直し

- 月1回程度アラート設定を見直す
- 販売実績に基づいて閾値を調整
- 不要なアラートは無効化

## アラートテンプレート

### 標準テンプレート

nancoでは以下の標準テンプレートを提供しています：

- **小売店向け**: 一般的な小売店の在庫管理
- **飲食店向け**: 食材の期限管理重視
- **製造業向け**: 原材料の在庫管理
- **ECサイト向け**: オンライン販売向け

### カスタムテンプレート

独自のアラートテンプレートを作成できます：

1. 「設定」→「アラートテンプレート」を選択
2. 「新規テンプレート作成」をクリック
3. テンプレート名と条件を設定
4. 「保存」をクリック

## アラートの無効化

### 一時的な無効化

1. 商品詳細画面で「アラート設定」をクリック
2. 該当するアラートの「無効」をクリック
3. 期間を指定して一時停止

### 完全な削除

1. 商品詳細画面で「アラート設定」をクリック
2. 該当するアラートの「削除」をクリック
3. 確認ダイアログで「削除」をクリック
